# 🧠 Hippocampus - 智能刷题系统

**Hippocampus（海马体）** 是一个基于 Django 的在线刷题平台，专为学生和考生设计。它能够自动解析试卷文件，生成可交互的在线题库，并智能记录错题，帮助用户高效复习。

---

## ✨ 核心功能

### 1. 📤 **智能试卷上传与解析**
- 支持上传 `.txt` 格式的试卷文件（未来可扩展 `.docx`）
- 自动解析试卷内容，识别题目类型（单选、多选、判断、简答、填空）
- 自动提取题号、题目内容、选项、答案、解析和分值
- 使用正则表达式智能识别试卷结构

### 2. 📝 **在线刷题体验**
- 逐题答题模式，支持题目导航
- 实时显示答题进度
- 支持多种题型：
  - 单选题
  - 多选题
  - 判断题
  - 简答题
  - 填空题
- 答题后即时显示正确答案和详细解析

### 3. 📊 **智能进度追踪**
- 自动保存用户答题进度
- 支持中途退出，下次继续答题
- 每道题记录用户答题状态（未答、已答、答对、答错）

### 4. 📖 **错题本功能**
- 自动记录答错的题目
- 统计每道题的错误次数
- 支持添加个人笔记
- 可以单独查看所有错题，针对性复习
- 支持收藏/取消收藏错题

### 5. 👥 **用户系统**
- 用户注册与登录
- 每个用户独立的答题记录和错题本
- 试卷归属管理（用户只能删除自己上传的试卷）

### 6. 🗑️ **试卷管理**
- 试卷列表查看
- 试卷预览（查看所有题目和答案）
- 删除试卷功能

## 🏗️ 技术架构

### 后端
- **框架**: Django 5.0+
- **数据库**: SQLite（开发环境），可轻松迁移到 PostgreSQL/MySQL
- **Web 服务器**: Gunicorn（生产环境）
- **静态文件**: WhiteNoise（用于生产环境静态文件服务）

### 前端
- **模板引擎**: Django Templates
- **样式**: 原生 CSS（现代化设计）
- **交互**: 原生 JavaScript（无框架依赖）

### 核心模块
1. **Parser 模块** (`core/parser.py`)
   - 负责解析上传的试卷文件
   - 使用正则表达式提取题目结构
   - 支持多种编码格式（UTF-8, GBK）

2. **Models 模块** (`core/models.py`)
   - `ExamPaper`: 试卷模型
   - `Question`: 题目模型
   - `UserProgress`: 用户答题进度
   - `UserMistake`: 错题记录

3. **Views 模块** (`core/views.py`)
   - 试卷上传与管理
   - 答题界面
   - 错题本
   - 用户认证
   - RESTful API 端点（进度同步、答案提交）

## 📦 数据模型

### ExamPaper（试卷）
- 标题
- 源文件
- 上传者
- 创建时间

### Question（题目）
- 所属试卷
- 题号
- 题型（单选/多选/判断/简答/填空）
- 题目内容
- 选项（JSON 格式）
- 参考答案
- 解析
- 分值

### UserProgress（用户进度）
- 用户
- 试卷
- 当前进度（题目索引）
- 最后更新时间

### UserMistake（错题记录）
- 用户
- 题目
- 错误次数
- 最近错误时间
- 错题笔记

## 🚀 快速开始

### 环境要求
- Python 3.8+
- pip

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/你的用户名/Hippocampus.git
   cd Hippocampus
   ```

2. **创建虚拟环境**
   ```bash
   python -m venv .venv
   .venv\Scripts\activate  # Windows
   # source .venv/bin/activate  # macOS/Linux
   ```

3. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

4. **初始化数据库**
   ```bash
   python manage.py migrate
   ```

5. **创建管理员账号**（可选）
   ```bash
   python manage.py createsuperuser
   ```

6. **启动开发服务器**
   ```bash
   python manage.py runserver
   ```

7. **访问应用**
   - 打开浏览器访问: `http://127.0.0.1:8000`
   - 管理后台: `http://127.0.0.1:8000/admin`

## 📋 使用指南

### 1. 注册/登录
首次使用需要注册账号，已有账号直接登录。

### 2. 上传试卷
- 点击"上传试卷"
- 输入试卷标题
- 选择试卷文件（`.txt` 格式）
- 系统会自动解析并创建题库

### 3. 开始刷题
- 在试卷列表中选择一份试卷
- 点击"开始答题"
- 逐题作答，系统自动保存进度

### 4. 查看错题
- 点击"错题本"
- 查看所有答错的题目
- 可以添加笔记辅助记忆

### 5. 预览试卷
- 在试卷列表点击"预览"
- 查看完整试卷和所有答案

## 📁 试卷文件格式

系统支持的 `.txt` 文件格式示例：

```
1. (单选题, 5.0 分) 以下哪个是 Python 的特点？
A 编译型语言
B 解释型语言
C 汇编语言
D 机器语言
答案：B
解析：Python 是一种解释型、面向对象的高级编程语言。

2. (多选题, 10.0 分) 以下哪些是 Web 框架？
A Django
B Flask
C NumPy
D FastAPI
答案：ABD
解析：NumPy 是科学计算库，不是 Web 框架。
```

**格式要求**：
- 题号格式：`数字. (题型, 分值)`
- 选项格式：`字母 内容`
- 答案格式：`答案：答案内容`
- 解析格式：`解析：解析内容`
